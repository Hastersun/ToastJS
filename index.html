The Unicorn-test Framework

What is the Unicorn-test Framework?

Unicorn-test is a easy to use Javascript testing framework. Unicorn-test has no dependencies library and supports all modern web browsers.

Browser Support

All of the modern browsers.

What can it do to help you?

It can help you to very easy to test your project.
